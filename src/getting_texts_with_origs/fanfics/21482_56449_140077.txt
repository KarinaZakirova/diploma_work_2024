Пополам.
Повдоль.
Нежные глаза Неллиэл теперь на разных половинах. Тонкие руки Неллиэл теперь на разных половинах. Стройные ноги Неллиэл теперь на разных.
Половинах.
Две Неллиел. Каждой по глазу. Кровавому мертвому глазу. По вытекшему глазу.
Каждой по руке. По сломанной руке. По сломанной ноге.
Две половины Неллиэл — ровные, точные, четкие. Ноитора старался. Санта-Тереза старалась. Меткие. Аккуратные.

Слабый слепой Тесла, новорожденный щенок Тесла, глупый, бестолковый, умирающий Тесла. Ест с рук. Дышит тепло на ладони.
Сердце Неллиэл.
Кожу Неллиэл.
Кости Неллиэл.
Слизывает кровь Неллиэл. Вдыхает смерть Неллиэл. Улыбается вкусу Неллиэл.
Куском чистой ткани вытирает рот Теслы. Старательный, предусмотрительный, заботливый Ноитора.
Тесла сыт, свернулся мягко, уютно, крепко. Довольный. Ноитора старался. Санта-Тереза старалась. Внимательные. Бдительные.

— Третий теперь я.
— Ноитора-сама? А... Неллиэл?

Жестокий. Страшный. Свирепый.
