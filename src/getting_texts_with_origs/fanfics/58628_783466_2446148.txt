По ночному ковылю,
По серебряной росе,
Во лугах встречать рассвет
По ночному ковылю.
По ночному ковылю,
За спиной оставив день.
Под ногами бьётся тень
По ночному ковылю.
По ночному ковылю,
Да по зеленым холмам,
Сквозь полуночный дурман
По ночному ковылю.
По ночному ковылю
Сбруей лёгкою звеня.
Не придерживай коня
По ночному ковылю.
По ночному ковылю
Рохан весь перед тобой
Пред тобой и под звездой
По ночному ковылю.
По ночному ковылю
Да по ветру напрямик.
Только пыль из-под копыт
По ночному ковылю.
По ночному ковылю
По свободе удалой,
По надежде молодой,
По ночному ковылю.
По ночному ковылю
Да привстав на стременах,
Разметав невольный страх,
По ночному ковылю.
По ночному ковылю,
Да по роханским полям
По безбрежным по степям
По ночному ковылю.
По ночному ковылю
Сбруей лёгкою звеня.
Не придерживай коня
По ночному ковылю.
